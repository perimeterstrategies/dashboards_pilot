# dashboards_pilot

Streamlit dashboard for analyzing Canadian productivity, GDP, labour, and capital trends using Statistics Canada industry data.

## Data source

Statistics Canada. Table 36-10-0208-01  Multifactor productivity, value-added, capital input and labour input in the aggregate business sector and major sub-sectors, by industry

Source table: https://www150.statcan.gc.ca/t1/tbl1/en/tv.action?pid=3610020801

## Running locally

1. Add `3610020801-eng.csv` to the repository root.
2. Install the required Python packages, including `streamlit`, `pandas`, and `plotly`.
3. Run `streamlit run app.py`.
